Make an application

Select application

When you request a change to your case, or contact the court to take a specific action, this is known as making an ‘application’ to the court for an order.

An application is not a new claim. It is a request for a judge to take an action on an existing claim.

Select the application you want and click continue.

If you need to make multiple applications, you’ll have the option to add another application later on
